Facebook Font Changer

Guide: Live Typing

1. Set Your Style First

Use the dropdown menu to choose the Unicode font you wish to type in live.

2. Type to Convert

New characters typed in the text area will be converted to the selected style instantly.

Guide: Existing Text

Highlight the text you want to change.

Select a new style or click [Reset to Normal].

Click the [Apply to Selection] button.
